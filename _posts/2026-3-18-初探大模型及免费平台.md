# Qwen3
## 快捷使用
```
# Use a pipeline as a high-level helper
from transformers import pipeline

pipe = pipeline("text-generation", model="Qwen/Qwen3-4B")
messages = [
    {"role": "user", "content": "Who are you?"},
]
pipe(messages)
```
使用 Hugging Face Transformers 库的 pipelineAPI​ 来调用 Qwen-4B 模型进行文本生成的标准方式
- pipeline是一个高级封装接口，让用户无需深入了解模型加载、预处理、推理等底层细节
- 任务类型："text-generation"（文本生成）、"text-classification"（文本分类）、"translation"（翻译）、"summarization"（摘要）等
- 每个消息是一个字典，包含：
    - role：角色，可以是 "user"（用户）、"assistant"（助手）、"system"（系统）
    - content：消息内容
- 管道内部会：
    - 将消息格式化为模型可接受的输入格式
    - 调用模型进行推理
    - 生成回复文本
    - 返回结果  

## 底层逻辑调用
```
# Load model directly
from transformers import AutoTokenizer, AutoModelForCausalLM

tokenizer = AutoTokenizer.from_pretrained("Qwen/Qwen3-4B")
model = AutoModelForCausalLM.from_pretrained("Qwen/Qwen3-4B")
messages = [
    {"role": "user", "content": "Who are you?"},
]
inputs = tokenizer.apply_chat_template(
	messages,
	add_generation_prompt=True,
	tokenize=True,
	return_dict=True,
	return_tensors="pt",
).to(model.device)

# outputs = model.generate(**inputs, max_new_tokens=40)
outputs = model.generate(
    **inputs,
    max_new_tokens=100,
    do_sample=True,              # 使用采样而非贪婪解码
    temperature=0.7,            # 控制随机性（0.1~1.0）
    top_p=0.9,                  # 核心采样（nucleus sampling）
    repetition_penalty=1.1,     # 减少重复
    pad_token_id=tokenizer.eos_token_id  # 设置填充token
)
print(tokenizer.decode(outputs[0][inputs["input_ids"].shape[-1]:]))
```

**解码输出**
- outputs[0]：获取批次中的第一个样本
outputs的形状是 (batch_size, sequence_length)
这里批次大小为 1，所以取索引 0
- inputs["input_ids"].shape[-1]：获取输入序列的长度
inputs["input_ids"].shape是 (batch_size, seq_len)
[-1]取最后一个维度，即序列长度
例如，如果输入是 20 个 token，这个值就是 20
- outputs[0][inputs["input_ids"].shape[-1]:]：提取生成的部分
从第 20 个 token 开始到末尾，这样就跳过了输入部分，只获取模型生成的新内容
- tokenizer.decode()：将 token ID 转换回文本
解码生成的部分，会自动处理特殊 token（如 <|im_end|>）
