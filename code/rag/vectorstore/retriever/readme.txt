## 参数配置建议
### alpha 参数配置
alpha 参数控制向量检索和 BM25 检索的权重：

- alpha = 0.0 ：完全使用 BM25 检索结果
- alpha = 0.5 ：向量检索和 BM25 检索权重相等（默认）
- alpha = 1.0 ：完全使用向量检索结果
配置建议 ：

- 对于关键词精确匹配场景： alpha = 0.3-0.4
- 对于语义理解场景： alpha = 0.6-0.7
- 对于混合场景： alpha = 0.5
### fusion_method 参数配置
fusion_method 参数控制两种检索结果的融合方式：

- "rrf" ：基于排名的融合（默认）
- "weighted" ：基于得分的加权融合
配置建议 ：

- 当两种检索方法的得分尺度差异较大时，使用 "rrf"
- 当您希望明确控制两种检索方法的权重时，使用 "weighted"
- 对于大多数场景，默认的 "rrf" 通常效果良好
